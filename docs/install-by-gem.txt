# Fluentd Ruby Gem Installation

This article describes how to install Fluentd from Ruby gem.

## Step1: Install Ruby interpreter

Please install Ruby >= 1.9.2 on your local environment.

## Step2: Install fluentd gem

Fetch and install the  `fluetnd` Ruby gem by `gem` command. Official ruby gem page is [here](https://rubygems.org/gems/fluentd).

    :::term
    $ gem install fluentd --no-ri --no-rdoc

## Step3: Run

To confirm that Fluentd was installed successfully, run the following commands:

    :::term
    $ fluentd --setup ./fluent
    $ fluentd -c ./fluent/fluent.conf -vv &
    $ echo '{"json":"message"}' | fluent-cat debug.test

The last command sends Fluentd a message ‘{“json”:”message”}’ with a “debug.test” tag. If the installation was successful, Fluentd will output the following message:

    :::term
    2011-07-10 16:49:50 +0900 debug.test: {"json":"message"}

NOTE: It's HIGHLY recommended to setup <b>ntpd</b> on the node. Otherwise your logs could contain invalid timestamps. Also for the large deployment, you must use <a href="http://www.canonware.com/jemalloc/">jemalloc</a> to avoid memory fragmentations. <a href="install-by-rpm">rpm</a> / <a href="install-by-deb">deb</a> packages already include it.

NOTE: fluentd gem doesn't come with /etc/init.d/ scripts. You should use the process management tools like <a href="http://cr.yp.to/daemontools.html">daemontools</a>, <a href="http://smarden.org/runit/">runit</a>, <a href="http://supervisord.org/">supervisord</a>, <a href="http://upstart.ubuntu.com/">upstart</a>.
