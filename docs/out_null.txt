# null Output Plugin

The `null` output plugin just throw away events.

### Example Config

`out_null` is included within Fluentd core. No extra installation process is required.

    :::text
    <match pattern>
      type null
    </match>

NOTE: See <a href="config-file">Config File</a> for the basic structure and syntax of the configuration file.

### Parameters

#### type (required)
The value must be `stdout`.
